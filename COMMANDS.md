This file is meant to be a reference to useful commands for the Portfolio project; Copy and Paste is king sometimes.

* Development Commands:
# Navigate to svelte folder
cd svelte

# Start development server
pnpm dev

# Build for production
pnpm build

# Test
pnpm test

# Preview production build
pnpm preview

# Install dependencies
pnpm install

* Git Scripts:
# Check status
clear; git status

# Commit and push
clear; git add . ; git commit -m ""; git push; git status

# View log
clear; git log --oneline

* Deployment Commands:
# Build and deploy (Vercel)
vercel --prod

# Preview deployment
vercel
